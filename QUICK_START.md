# 🚀 MCQ Platform - Quick Start

## You're All Set! Here's what to do:

### Step 1: Start the Server (if not already running)
```bash
npm run dev
```

### Step 2: Go to Setup Page
Open: **http://localhost:3000/setup**

### Step 3: Seed Sample Courses
Click the **"Seed Database"** button to add 6 sample courses instantly.

### Step 4: Create Account
- Go to **http://localhost:3000/register**
- Create an account with any email and password
- You'll be logged in automatically

### Step 5: Become an Admin (Optional)
1. Open MongoDB (Atlas or local)
2. Go to `aimcq` database → `users` collection
3. Find your user → Change `role` to `"admin"`
4. Refresh the page

### Step 6: Access Admin Panel
Go to **http://localhost:3000/admin** to manage everything!
